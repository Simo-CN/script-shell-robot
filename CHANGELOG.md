### Version 1.0.0
SSR - Script Shell Robot的初代版本
支持以下特性：
1. 结构化单行命令脚本，赋能网络设备自动化配置流程，支持使用动态配置参数。
2. 基于xterm的终端页面，给予运维最原始的终端体验
3. 集成OpenAI SDK的Python后端，并提供原生工具，允许AI在监管下执行命令、查看信息，提高配置和排查效率
4. 使用二级安全审核机制，对AI请求执行的命令采用安全分级方式，提高自动化管控能力

### Version 1.0.1
修复windows环境下，弹出对话框再关闭导致的无法聚焦输入框的问题

### Version 1.0.2
修复windows环境下默认字符集导致使用OpenAI SDK时出现的编码格式报错问题

### Version 1.1.0
该版本做出以下修改：
1. 优化应用图标
2. 修改工具调用信息卡片：
    - 允许折叠详细信息（默认折叠）
    - 移除Emoji显示，更清爽
    - 修复宽度随聊天气泡宽度变动的问题
3. 增加终端个性化设置：
    - 允许设置背景颜色
    - 允许设置字体颜色
    - 允许设置字体大小
4. 增加二级审核严格调整机制：
    允许用户手动调整需要人工校验的最低安全等级，可根据需求给予AI权限
5. 增加二级审核功能的历史命令队列审核：
    防止出现连续执行的命令导致系统故障、重启、连接丢失等致命问题
6. 优化AI聊天气泡中，表格和代码块的样式，允许放大查看详细代码和表格
7. 修复会话管理中出现的问题：
    - 会话更新时间为最后打开时间而不是完成时间
    - 会话排序按照最后打开时间排序
    - 点击展开更多会直接关闭下拉框
8. 打开设置界面时默认为AI模型界面而非通用界面
